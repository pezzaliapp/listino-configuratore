Admin Promo

Gestione senza Cloudflare: modifichi data validità, disattivi o cancelli una promo, poi scarichi il nuovo promo.json e lo carichi su GitHub nella cartella promo/.

Accesso amministratore

Password predefinita: admin. Puoi cambiarla dentro questo file, ma su GitHub non è una protezione server reale.

Promo nel file promo.json

Per cancellarla dalle app: premi Disattiva o Elimina, poi scarica il file e sostituisci promo/promo.json su GitHub. Gli utenti vedranno l’aggiornamento riaprendo la PWA o premendo “Aggiorna”.
Nota: se vuoi rimuovere anche il file PDF/JPG fisico, cancellalo dalla cartella promo/ su GitHub. Se basta non mostrarlo più, è sufficiente disattivarlo o eliminarlo dal JSON.
