Admin Promo

Gestione senza Cloudflare: puoi caricare un promo.json, modificare data inizio/scadenza, disattivare o cancellare una promo, poi scarichi il nuovo file e lo sostituisci su GitHub nella cartella promo/.

Accesso amministratore

Password predefinita: admin. Puoi cambiarla dentro questo file. Senza backend non è una protezione server reale.

Torna alla app
Importante: questa pagina non può scrivere direttamente su GitHub. Le modifiche vengono salvate localmente su questo browser e nel JSON qui sotto. Per aggiornare tutte le app devi premere Scarica promo.json e sostituire promo/promo.json su GitHub.
Ora il salvataggio funziona: ogni modifica, Disattiva, Riattiva o Elimina viene salvata subito in locale. Se esci e rientri in Admin la ritrovi. Poi scarichi il JSON definitivo.

Carica / importa JSON

Se hai un JSON nuovo, caricalo qui. Verrà usato come base per modifiche, date e cancellazioni.
